# 2026-04-06 - Actualización de changelog de release

## Qué cambió

- Se agregó al `CHANGELOG.md` un nuevo bloque superior para la versión `SISOC 06.04.2026`.
- El bloque nuevo cubre solo los cambios posteriores al release ya documentado de `01.04.2026`, evitando duplicar lo que ya estaba registrado.
- Se consolidaron en el resumen los cambios de VAT, importaciones, accesos y correcciones de migración que entraron entre el `02.04.2026` y el `06.04.2026`.

- Se mantuvo el formato existente del changelog en tres categorías: `Nuevas Funcionalidades`, `Actualizaciones` y `Corrección de Errores`.

- Se tomó como referencia el historial de `docs/registro/cambios/` para no repetir cambios ya absorbidos en la última versión.
