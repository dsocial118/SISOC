# VAT: alta de curso desde plan curricular en detalle de centro

- Se agregó una acción `+` en la tabla de planes curriculares de `vat/centros/<id>/#cursos`.
- El botón abre el mismo modal de `Nuevo Curso` y precarga `Plan de Estudio` con el plan seleccionado.
- Cuando el alta parte desde esa tabla, el campo `Plan de Estudio` queda bloqueado para evitar cambiar la relación elegida.
