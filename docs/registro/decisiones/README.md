# Registro de Decisiones Importantes

Guardar aquí decisiones de producto/técnicas que condicionen implementaciones futuras.

Formato recomendado por archivo:

```md
# YYYY-MM-DD - <titulo de la decision>

## Estado
- propuesta | aceptada | reemplazada

## Contexto
- Qué situación requiere decidir.

## Decision
- Qué se decide exactamente.

## Consecuencias
- Impacto positivo.
- Costos o trade-offs.

## Referencias
- Links a PRs, issues o docs relacionadas.
```
